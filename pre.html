01. 문제 정의

"제주에 한 달 살러 왔는데,
오늘 뭐하지?"

여행은 며칠이면 끝나지만, 한달살이는 이 고민을 서른 번 해야 해요. 그런데 기존 정보로는 답을 찾기 어려워요.

cloudy_snowing

변덕스러운 제주 날씨

제주시는 맑은데 애월은 비가 와요. 동네마다 날씨가 달라서, 일기예보만 믿고 나갔다가는 낭패 보기 일쑤예요.

history

오래된 관광 정보

검색하면 몇 년 전 관광 후기만 나와요. 지금 주차가 되는지, 얼마나 붐비는지는 아무도 알려주지 않아요.

quiz

매일 반복되는 선택 피로

한 달 내내 매일 갈 곳을 직접 정해야 해요. 결국 갔던 곳만 또 가거나, 숙소에만 머물게 돼요.

제주에서의 하루하루를
현지의 실시간 정보로 채웁니다.

제주옵서는 지금 제주에 살고 있는 사람들이 서로 알려주는 앱이에요. 내가 본 하늘을 사진으로 제보하고, 다른 사람의 꿀팁을 보고 오늘 갈 곳을 정해요.

groups

서로 알려주는 동네 소식

지금 그 동네에 있는 사람이 사진으로 날씨를 알려주고, 주차·가격·붐빔 같은 꿀팁을 남겨요.

cloud_sync

빈 동네는 기상청 날씨로

아직 제보가 없는 동네는 기상청 날씨로 자동으로 채워요. 어느 동네를 봐도 빈칸이 없어요.

my_location

앱이 갈 곳을 골라줘요

내가 지금 있는 곳과 오늘 날씨에 맞춰서, 갈 만한 곳을 앱이 대신 골라줘요.

03. 오늘의 추천 홈

홈 화면

앱을 열면
답이 나온다

앱을 열면 지금 내가 있는 동네를 알아채고, 오늘 날씨와 거리까지 따져 갈 만한 곳 3곳을 골라줘요. 마음에 안 들면 '다른 추천 보기'로 얼마든지 새로 받을 수 있어요.

lock

내 위치는 추천에만 잠깐 쓰고, 어디에도 저장하거나 남기지 않아요.

thermostat
첫째 · 오늘 날씨부터 확인

근처에서 올라온 실시간 제보로 지금 날씨를 확인해요. 비가 오면 실내 장소를 먼저 보여줘요.

done_all
둘째 · 안 가본 곳 위주로

이미 가 본 곳은 빼고 골라줘요. 한 달 내내 매일 새로운 곳을 만날 수 있어요.

near_me
셋째 · 가까운 곳 먼저

지금 있는 곳에서 가까운 장소부터 보여줘서, 오가는 부담이 없어요.

tips_and_updates
넷째 · 경험담 많은 곳 먼저

가 본 사람들의 꿀팁이 많이 쌓인, 믿을 만한 장소를 먼저 추천해요.

diversity_3
다섯째 · 내 또래가 좋아한 곳

나와 비슷한 연령대가 많이 찾은 장소를 조금 더 챙겨서, 취향에 맞는 추천이 돼요.

04. 실시간 날씨 제보 & 지역 피드

사진 한 장 · 10초 제보
add_a_photo

원터치 날씨 제보

하늘 사진 한 장 찍고, 날씨 버튼(맑음·흐림·비 등)을 누르면 끝이에요. 장소는 지도에서 이름만 검색해 고르면 나머지는 앱이 알아서 채워줘요.

gps_fixed

진짜 현장에서만 — 실제로 그 장소에 있을 때만 제보할 수 있어요. 다른 곳이나 옛날 사진은 올릴 수 없어요

timer

2시간이 지나면 사라져요 — 오래된 날씨가 남지 않아서 언제 봐도 '지금' 날씨예요

map

읍면동 단위 실시간 피드

애월읍부터 대정읍까지 제주 13개 동네의 지금 하늘을 한눈에 봐요. 제보가 있는 동네는 현장 사진과 함께, 없는 동네도 빈칸 없이 보여줘요.

cloud_download

빈 동네는 기상청이 채워요 — 제보가 없는 동네도 기상청 공식 날씨와 기온이 자동으로 나와요

cached

항상 최신 날씨 — 날씨 정보를 수시로 새로 받아와서 언제 봐도 지금 상태예요

verified

그래서 믿을 수 있어요 — 그 자리에 있는 사람만 올릴 수 있으니, 화면 속 모든 사진이 '진짜 지금 그곳'이에요.

06. 이렇게 만들었어요 · 사용 기술

code

Django 웹앱

파이썬 Django로 서버부터 화면까지 한 번에 만들었어요. 설치 없이 주소만 입력하면 열리고, 홈 화면에 추가하면 일반 앱처럼 쓸 수 있어요.

Django Python PWA (홈 화면 설치)
api

공공·카카오 API 연동

기상청 단기예보 API로 실시간 날씨를 받아오고, 카카오맵 API로 실제 등록된 장소만 검색해요. 제보할 때는 GPS로 현장 여부를 확인해요.

기상청 API 카카오맵 API GPS 위치 인증
cloud_upload

Value & Impact
Why it works

현장이 만드는
선순환 구조

사용자가 곧 정보 제공자가 되는 커뮤니티 구조. 제보가 쌓일수록 추천이 정확해지고, 정확한 추천이 다시 제보를 부릅니다.

photo_camera
STEP 01

현장 제보로 데이터가 쌓인다

현장에서 인증된 실시간 날씨와 꿀팁이 동네별로 쌓여요. 오래된 정보는 저절로 사라져 늘 신선해요.

insights
STEP 02

추천이 정교해진다

정보가 쌓일수록 "지금 나에게 맞는 장소"를 더 정확하게 골라줘요. 안 가본 곳 위주라 매일 새로워요.

favorite
STEP 03

한달살이 만족도가 올라간다

실패 없는 나들이 경험이 쌓이고, 만족한 사용자가 다시 제보하며 커뮤니티가 성장하는 선순환이 완성됩니다.
